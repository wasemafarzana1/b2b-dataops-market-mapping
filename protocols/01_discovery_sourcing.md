# Protocol 01: Sourcing Parameters & Regional Stratification

## 1.1 Objective
To segment sourcing parameters across two primary geolocations (US and India) within the Talent Acquisition and HR Tech infrastructure ecosystems.

## 1.2 Regional Density Mapping
The production run yields the following target distribution across nodes:
* **United States (US):** 54 nodes optimized for enterprise platform evaluation and sourcing design.
* **India (IN):** 46 nodes optimized for high-velocity software engineering recruitment frameworks and regional delivery management.

## 1.3 Boundary Scope
Any node entering this system must map cleanly to one of these predefined operational profiles. Generalist human resource generalists are excluded; targets must explicitly demonstrate domain expertise in automated tooling, sourcing algorithms, technical recruiting networks, or work-tech SaaS advisory roles.
